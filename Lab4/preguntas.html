JavaScript:

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Diferencias

    • Tipo de lenguaje: JS es un lenguaje interpretado, mientras que Java es un lenguaje particular porque es compilado, pero es compilado a un lenguaje intermedio llamado bytecode, que después es interpretado.
    • Tipado: JavaScript tiene un tipado débil y Java presenta un tipado fuerte.
    • Proposito: Mientras que Js esta recomendado para aplicaciones y páginas web, Java es un lenguaje de programación originalmente diseñado para aplicaciones integradas.

  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    • getDate(): Devuelve el día del mes. Número entre 1 y 31.
    • getDay(): Devuelve el día de la semana. Entre 0 (domingo) y 6 (sábado).
    • getFullYear(): Devuelve el año con 4 dígitos.
    • getMonth(): Devuelve el mes. Entre 0 (enero) y 11 (diciembre).
    • toTimeString(): Convierte la parte de tiempo de un objeto Date en una cadena.

  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    • find(): Recorre el array y retorna la primera coincidencia del elemento que se busca.
    • sort(): Ordena los elementos del array y retorna el arreglo ordenado.
    • push(): Añade uno o más elementos al final de un array.
    • pop(): Elimina el último elemento de un array y lo devuelve.
    • map(): Permite recorrer el array y modificar los elementos presentes en él, retornando un nuevo array con la misma longitud que el original.

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    • let es una palabra reservada que se utiliza para declarar variables locales o block scoped variables.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Si declaramos la variable con var también dentro de la función, se crea una variable local, se reduce a la función donde ha sido declarada, vat, a diferencia de let, permite redefinir la misma variable en el mismo bloque, esto puede provocar que se alteren los valores de una variable de forma accidental y genere errores o complicaciones difíciles de depurar.
