  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    • Cuando hablamos de internet nos referimos a todo el conjunto de redes y computadoras distribuidas alrededor del mundo que utilizan protocolos TCP/IP y que son compatibles entre sí; por otro lado la Word Wide Web es un sistema interconectado por el cual se pueden transmitir diversos tipos de datos a través del Protocolo de Transferencia de Hipertextos o HTTP que funciona a través de Internet. En resumen Internet es la infraestructura que proporciona los recursos necesarios para que la World Wide Web se sostenga.

  • ¿Cuáles son las partes de un URL?

    • Protocolo: Formato por el cual se transfieren o intercambian los datos del navegador y la máquina de destino.
    Host: Indica qué servidor web que contiene el recurso que se está solicitando. Se puede dividir en 3 partes:

    • Subdominio: Es una subclasificación o subgrupo del nombre del dominio que ayudan a organizar y navegar a diferentes secciones de tu sitio web.
    • DominIo: Nombre único que identifica al sitio web.
    • Extensión/Terminación: Es la parte final de una dirección web y que nos indica la naturaleza del mismo dominio.

    Ruta: Identifica la ubicación del recurso web al que se desea acceder.
    Parámetros: Es la información adicional usada para especificar información del recurso para algún propósito.

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • GET: Recupera o lee un recurso.
    • HEAD: Es similar al método HTTP GET, pero no retorna ningún contenido HTTP Response, solo el código de respuesta y los encabezados.
    • POST: Crea un recurso o un conjunto de recursos.
    • PUT: Reemplaza/Actualiza un recurso o un conjunto de recursos.
    • PATCH: Modifica parcialmente a un recurso.
    • DELETE: Elimina un recurso o un conjunto de recursos.

  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?
